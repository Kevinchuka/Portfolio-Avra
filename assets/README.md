# Media assets

Esta carpeta contiene rutas sugeridas y ordenadas para medios del sitio.

## Imágenes
- `assets/images/hero/` — banners y portadas principales
- `assets/images/projects/` — capturas de proyectos
- `assets/images/profile/` — fotos de perfil y bio
- `assets/images/logos/` — logotipos y marcas
- `assets/images/icons/` — íconos e ilustraciones pequeñas
- `assets/images/backgrounds/` — fondos y texturas
- `assets/images/blog/` — imágenes para artículos
- `assets/images/ui/` — elementos de interfaz

## Videos
- `assets/videos/hero/` — videos principales o headers
- `assets/videos/demos/` — demos de productos o proyectos
- `assets/videos/backgrounds/` — videos de fondo
- `assets/videos/testimonials/` — testimonios
