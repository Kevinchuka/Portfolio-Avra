# Media

Estructura recomendada para activos estáticos:

- `public/media/images/` → Imágenes del sitio (logos, fondos, fotografías, etc.).
- `public/media/videos/` → Videos promocionales o de fondo.

Coloca los archivos finales aquí para que estén disponibles en rutas públicas como:

- `/media/images/nombre-del-archivo.ext`
- `/media/videos/nombre-del-archivo.ext`
